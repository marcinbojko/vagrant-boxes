## Vagrant required plugins

```bash
vagrant plugin install sahara
vagrant plugin install vagrant-reload
```

## Default credentials

### Linux

```ini
root:password
vagrant:vagrant
```

### Windows

```ini
Administrator:password
vagrant:vagrant
```
